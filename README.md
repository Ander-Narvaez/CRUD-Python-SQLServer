# CRUD-Python-SQLServer  📦

 * Descripción 📖  
 
  Sistema web en Python CRUD, almacena la información en una base de datos relacional con SQlServer.  
  Está implementado con las siguientes tecnologías:
  * Boostrap v4.5 
  * Flask v2.1.2
  * Flask-Login v0.6.1
  * pyodbc v4.0.32
  * SQLAlchemy v1.4.39
  * SweetAlert2
  * JavaScript
